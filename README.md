# Capela São João Bosco - Site

## Descrição
Este é um site desenvolvido para a Capela São João Bosco, com o objetivo de apresentar informações importantes como horários de missa, adoração, contato e localização.

## Objetivo do site
O projeto foi criado para praticar HTML e CSS, além de oferecer uma solução simples e útil para divulgar as informações da capela.

## Funcionalidades
- Informações sobre a capela
- Horários de missas e adoração
- Link para o grupo do whatsapp
- Localização com mapa integrado

## Tecnologias Utilizadas
- HTML5
- CSS3
